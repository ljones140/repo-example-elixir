# Repo example for Elixir

**An example open source Elixir application.**

## Installation

Add `repo_example` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:repo_example, "~> x.y.z"}
  ]
end
```
## Documentation

The documentation can be found on [HexDocs](https://hexdocs.pm/repo_example).
